`Multi LLM Chat Interface`
This repository provides a simple interface (that can be extended) to chat with various different LLM models. The ones currently supported include: GPT-4, Claude Opus, and Gemini Pro

```Setup```
- Clone the project to your machine with git clone
- Navigate to the backend folder and set your API keys
- Start the backend using the yarn start command
- Navigate to the frontend and 'go live' using LiveServer
